# fta-dumi-vue

> 这是一个探索性 demo 项目  
> 旨在实验基于 `dumi` 文档框架 展示 Vue 组件的效果

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
